=======================
Umbraco 中文介紹說明
=======================

Umbraco 是一個以 Asp.Net 開發的\ `內容管理系統`_，有著簡單美觀的操作介面，方便的管理系統，而且是個免費的開放原始碼程式，目前已經進展到 7.5.x 的版本，穩定而且容易擴充，不過中文參考以及翻譯部分之前都較為缺乏，所以在這個地方向大家介紹，並分享中文化檔案。

而本書內容版本則和 Umbraco 主版本互相對應，讓使用者較方便參考。如果有建議，歡迎在下方留言或到 GitHub 提出 Pull-request。
